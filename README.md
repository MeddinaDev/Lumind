# ⚡️ Lumind

> Asistente integral de productividad y bienestar digital.

## Sobre el Proyecto

Lumind es una aplicación móvil multiplataforma desarrollada en Flutter que busca solucionar la fragmentación del flujo de trabajo y reducir la fatiga digital. Diseñada bajo el concepto de "Super-app" de nicho, unifica las herramientas esenciales para el estudio y el trabajo en un único ecosistema, fomentando la concentración y cuidando la salud mental del usuario a través de una experiencia de uso fluida y premium.

## Arquitectura y Tecnologías

El proyecto está construido priorizando la escalabilidad, la mantenibilidad y la estricta separación de responsabilidades, aplicando estándares de la industria para el desarrollo de software:

- **Framework:** Flutter / Dart
- **Backend & Auth:** Supabase (Autenticación segura y Base de Datos PostgreSQL en la nube)
- **Arquitectura:** Clean Architecture (Presentation, Domain, Data)
- **Gestión de Estado:** BLoC Pattern (Business Logic Component)
- **UI/UX Premium:** 
  - Diseño minimalista con transiciones fluidas de navegación.
  - *Haptic Feedback* (Respuesta táctil física del sistema).
  - Refuerzo auditivo de éxito en microinteracciones.
  - Animaciones implícitas para un control visual satisfactorio.

## Funcionalidades Core

- [x] **Focus Timer (Pomodoro):** Temporizador de precisión con retroalimentación visual de anillo dinámico, respuesta táctil y alertas sonoras inmersivas.
- [x] **Gestor de Tareas:** Sistema CRUD completo integrado con el motor Pomodoro y animaciones interactivas al completar objetivos.
- [x] **Dashboard de Progreso:** Panel de estadísticas conectadas en tiempo real a la nube para medir minutos de enfoque, sesiones completadas y progreso histórico.
- [x] **Control de Acceso:** Sistema de autenticación de usuarios con persistencia de sesión automática y encriptada.
